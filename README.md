# So9ne App (So9ne Fashion Marketplace)

So9ne is a fashion marketplace application built with Flutter.

## Tech Stack
- **Framework**: Flutter
- **State Management**: Riverpod (`flutter_riverpod`)
- **Routing**: GoRouter (`go_router`)
- **Backend as a Service**: Supabase (`supabase_flutter`)
- **Typography**: Google Fonts (`google_fonts`)
- **Localization**: Intl (`intl`)

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites
- Flutter SDK (version 3.11.1 or higher)
- Supabase Project configured

### Setup
1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Configure your Supabase URL and Anon Key in `lib/main.dart` or via environment variables (if configured).
4. Run the app using `flutter run`.
